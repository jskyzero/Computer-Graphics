Learn OpenGL from [LearnOpenGL CN](https://learnopengl-cn.github.io/)


## Overview
+ Install: 
  + see playground about how to set the playground environment
  + change the `CMakeLists.txt` file `OpenGL_PATH` to where we can find the library
+ Run: 
    ```Bash
    mkdir build
    cd build
    cmake ..
    make
    ```
